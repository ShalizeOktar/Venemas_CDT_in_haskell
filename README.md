# Venemas_CDT_in_haskell

During our Master's degree in Linguistics we attended the course "Functional programming and formal linguistics" at the HHU Düsseldorf by Dr. Patrick David Elliot. Within our course project we implemented the modal logic CDT by Venema (1991)[^1] in Haskell[^2].

CDT:
Next to the classic logical operators $\wedge$, $\vee$, $\neg$ and $\rightarrow$, the modal logical operators $Chop$, $Done$ and $To Come$ are added.

Implementation:
The implementation can be found in CDTinHaskell.hs and a detailed explanation in our paper.

`Version: GHC 9.12.2`

[^1]: YDE Venema. 1991. https://doi.org/10.1093/logcom/1.4.453 A Modal Logic for Chopping Intervals. Journal of Logic and Computation, 1(4):453–476
[^2]: https://www.haskell.org
